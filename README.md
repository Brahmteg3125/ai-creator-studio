# 🎬 AI Creator Studio

Create fictional AI creators who generate their own identity-consistent images, scripts, voice, and short lip-synced videos — an end-to-end generative-AI content pipeline.

## ✨ Features (in progress)
- Define a persona: name, look, personality
- Generate identity-consistent images (SDXL + ControlNet)
- Write scripts with an LLM
- Convert scripts to speech (Text-to-Speech)
- Lip-sync speech onto generated faces (Wav2Lip)
- Assemble short social-media-style videos

## 🛠️ Tech Stack
Python · Diffusers / SDXL · HuggingFace · Wav2Lip · TTS · FastAPI

## 🚀 Setup
```bash
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```
